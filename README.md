# Exponential Platform v4.6.x DXP Symfony Flex metapackage

This is a metapackage for Symfony Flex-based Exponential Platform v4.6.x DXP installation.

## Installation

To install the latest version of Exponential Platform v4 DXP, create a new project from [ibexa/oss-skeleton](https://github.com/ibexa/oss-skeleton):

```
composer create-project se7enxweb/exponential-platform-v4x-dxp-skeleton:^4.6
```

Learn more about our commercial product - [7x Exponential Platform DXP](https://se7enx.com/products).
